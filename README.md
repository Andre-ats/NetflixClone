# Netflix Clone

Projeto desenvolvido com o objetivo de **praticar desenvolvimento front-end**, recriando a interface da Netflix com foco em organização de componentes, estrutura visual e experiência do usuário.

A aplicação foi construída utilizando **React**, **JavaScript**, **HTML** e **CSS**, utilizando dados reais de filmes e séries através da API do **The Movie Database (TMDB)**.

## 🚀 Tecnologias

* React
* JavaScript
* HTML
* CSS
* TMDB API

## 📚 Sobre o Projeto

Este projeto foi criado como exercício de prática em **desenvolvimento front-end**, com foco na construção de interfaces modernas inspiradas em plataformas reais.

A proposta foi recriar a experiência visual da Netflix, trabalhando conceitos como **componentização em React**, organização de layout e exibição dinâmica de conteúdos.

Para simular um catálogo real de filmes e séries, o projeto utiliza a API pública do **The Movie Database (TMDB)**, permitindo carregar e exibir dados reais diretamente na interface.

API utilizada:

```
https://api.themoviedb.org/3
```

## ⚙️ Funcionalidades

* Reprodução visual da interface da Netflix
* Organização da interface com componentes em React
* Consumo de API para obtenção de filmes e séries
* Exibição dinâmica de conteúdos na página
* Estruturação de layout com HTML e CSS
